# 大连市贝雕非物质文化遗产的保护与传承

### 前言

本次调查聚焦大连市贝雕非物质文化遗产的保护与传承，基于227份有效问卷展开。问卷内容涵盖人口统计特征、消费习惯、对非遗的认知程度、对大连贝雕的兴趣及产品偏好等方面。调查初步揭示了公众对大连贝雕文创产品的潜在兴趣和消费意愿，为后续保护与传承策略提供了基础洞察。

## 问卷的信度很高

信度分析结果表明，问卷的Cronbach’s α系数为0.867，超过了常用的标准值0.8，表明问卷具有较高的内部一致性，能够稳定地测量相关的心理和行为变量。
